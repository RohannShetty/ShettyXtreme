Trading API Credentials
Enter your Dhan Trading API credentials to connect your trading account.
▶ How to get credentials from Dhan Developer Portal
  1. Go to https://developer.dhan.co/ and log in
  2. Click "Create New App" or select your existing app
  3. Copy your Client ID (a string like "11345...")
  4. Click "Generate API Key" and copy it
  5. In the same app settings, set the Postback URL to:
http://localhost:8000/api/postback/dhan
Step 2: Register OAuth Redirect URL
In your Dhan app settings, set the OAuth Redirect URL to:
http://localhost:8000/auth/dhan/callback
This is required so Dhan can redirect back to your terminal after you approve the consent. Without this, the authorization will appear stuck on "Pending Authorization" forever.
Your credentials are entered as: client_id:::api_key
Example: 11345:::abc123def456
Data API Credentials
Enter your Dhan Data API credentials for market data access.
▶ How to get Data API credentials
  1. In the Dhan Developer Portal, go to your app
  2. Scroll to the Data API section
  3. Copy the Data Client ID and Data API Key
Your credentials are entered as: client_id:::api_key
Register OAuth Redirect URL
Set the OAuth Redirect URL in your Dhan app settings to:
http://localhost:8000/auth/dhan/callback
The same URL is used for both Trading and Data API authorization flows. Without this, authorization will appear stuck on "Pending Authorization" forever.
