# Section 14: Delivery Roadmap
Phase 0 (2-3 wks): Architecture and repo strategy. Phase 1 (4-6 wks): Foundations - core interfaces, event bus, adapters, data pipeline. Phase 2 (6-8 wks): Usable MVP - terminal UI, execution cockpit, scanners. Phase 3 (6-8 wks): Advanced intelligence - regime engine, signals, options AI. Phase 4 (ongoing): Platform maturity - plugins, automation, SaaS.
